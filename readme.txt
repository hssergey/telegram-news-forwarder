Telegram News Forwarder - simple application to collect your favourite news telegram channels and forward their content to one selected chat - so you need to read one chat only to get all news.

System requirements:
- python 3.10 or newer
- python virtual enviroment

Installation:
- clone the repository
- rename settings.py.example into settings.py and edit configuration
- setup virtual enviroment:
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
- then run application by command  python main.py
